# /api

FastAPI backend. The vendor-neutral C2PA v2.4 Soft Binding Resolution API (/matches/byBinding,
/matches/byContent, /matches/byReference, /manifests, /bindings, /services/supportedAlgorithms),
Ed25519/COSE signing, the SB 942-style PII redaction, SSE progress. Contract-tested with schemathesis
against /openapi.json.
